## Rules of Engagement

### General Rules
1. Always be polite in all communications
2. Flag any payment over $500 for human approval
3. Never send bulk emails without approval
4. Keep all data local-first (privacy focused)
5. All sensitive actions require human approval

### Communication Rules

#### Email
- Respond to known contacts within 24 hours
- Flag urgent messages with priority tag
- Auto-reply to known contacts
- **New contacts require approval before reply**

#### WhatsApp
- Respond to messages containing keywords: "urgent", "asap", "invoice", "payment", "help"
- **All WhatsApp replies require approval before sending**
- Never send bulk messages without approval

### File Handling Rules
- Process all files in /Needs_Action/ folder
- Move completed items to /Done/
- Log all actions in /Logs/ folder
- Sensitive actions go to /Pending_Approval/ first

### Approval Thresholds (Silver Tier)
| Action | Auto-Approve | Require Approval |
|--------|-------------|------------------|
| Read emails/messages | ✅ Yes | - |
| Draft replies | ✅ Yes | - |
| Send emails | - | ❌ Always |
| WhatsApp replies | - | ❌ Always |
| Payments | - | ❌ Always |
| File operations | Create, Read | Delete, Move |

### Scheduling
- Daily briefing: 8:00 AM
- Weekly audit: Sunday 10:00 PM
- Continuous monitoring: Active during business hours
